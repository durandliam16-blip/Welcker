
## Explication du contenu de chaque fichiers

- index.html = contient les 11 pages
- app.js = toutes les fonctions du site
- styles.css = contient tout le style du site
- ex_visuel = dessin représentantr mon idée pour la pge d'acceuil

- sw.js = tourne en arrière-plan et permet de mettre en cache tes fichiers (index.html, styles.css, app.js) pour que l'application s'affiche instantanément, même sans connexion internet
- manifest.json = permet au navigateur de comprendre que ton site est une application. Il définit le nom, les couleurs et comment l'app doit s'ouvrir.

- icon-192.png = logo format 192x102pixels
- icon-512.png = logo format 512x512pixels
- logo = logo fond transparent
